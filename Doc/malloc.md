# 内存分配（堆）

## 主要功能

### 堆初始化`nano_crt_init_heap()`

- Windows下采用`VirtualAlloc()`直接申请空间，Linux下使用`brk()`将数据段结束地址向后调整从而得到一块空间

### 堆的分配`malloc()`

- 采用空闲链表算法进行分配

### 堆的释放`free()`

- 采用空闲链表法，若释放后可合并则会进行空闲块的合并

## 待做功能
